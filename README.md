# 📦 Portfolio Templates Bundle

A collection of clean, modern, and beginner-friendly portfolio templates.

This repository contains multiple portfolio projects, each in its own folder, designed to help junior developers learn, experiment, and build their own personal websites.

---

## 🎯 Purpose

The goal of this project is to provide:

* Ready-to-use portfolio templates
* Learning resources for frontend beginners
* Real examples of responsive layouts and UI design
* A base for customization and experimentation

---

## 📂 Structure

Each portfolio is separated into its own folder:

```id="p4k82x"
portfolio-templates-bundle/
│── portfolio 1/
│── portfolio 2/
│── README.md
```

Each folder contains a complete standalone project.

---

## ✨ What's Included

* Multiple portfolio designs
* Responsive layouts (mobile-friendly)
* Modern UI styles
* Different layout approaches
* Clean and readable code

---

## 🚀 How to Use

1. Clone the repository:

```bash id="t0m3va"
git clone https://github.com/thanos-coder2/portfolio-templates-bundle.git
```

2. Open any portfolio folder:

```bash id="d6lgqk"
cd portfolio-1
```

3. Open `index-en.html` in your browser

4. Customize it:

* Change content
* Replace images
* Modify styles
* Add your own projects

---

## 🧠 Who is this for?

* Junior developers
